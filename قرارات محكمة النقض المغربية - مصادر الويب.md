## 1. 🥇 juriscassation.cspj.ma — البوابة القضائية للمملكة المغربية (الأفضل)

### الوصف
المنصة الرقمية الرسمية لقرارات محكمة النقض، تابعة للمجلس الأعلى للسلطة القضائية (CSPJ). أطلقت في 26 يناير 2022.

### البيانات المتاحة
- **51,850 قرار** من محكمة النقض
- **4,524 قرار** من محاكم الاستئناف
- **966 حكم** من المحاكم الابتدائية
- **36 قرار مصحوب بتعاليق**
- 26 مليون+ زيارة

### الغرف المتاحة للبحث
| ID | الغرفة |
|----|--------|
| 1 | الغرفة المدنية |
| 2 | غرفة الأحوال الشخصية والميراث |
| 3 | الغرفة التجارية |
| 4 | الغرفة الإدارية |
| 5 | الغرفة الاجتماعية |
| 6 | الغرفة الجنائية |
| 7 | الغرفة العقارية |

### طريقة البحث البرمجي (مثالية للوكلاء الذكيين)

```bash
# الخطوة 1: الحصول على CSRF token وملفات تعريف الارتباط
curl -sL -c cookies.txt -H "User-Agent: Mozilla/5.0" \
  "https://juriscassation.cspj.ma/Decisions/RechercheDecisions" \
  | grep -o 'value="CfDJ[^"]*"' | head -1 | sed 's/value="//;s/"//'

# الخطوة 2: إرسال البحث (POST form)
curl -sL -b cookies.txt -H "Content-Type: application/x-www-form-urlencoded" \
  -d "NumeroDos=&NumeroDec=&DateDec=&ChambreIds=1&DecisionPriseParId=&Sujet=طلاق&__RequestVerificationToken=TOKEN" \
  "https://juriscassation.cspj.ma/Decisions/RechercheDecisionsRes"
```

### معايير البحث المتاحة
- `NumeroDos` — رقم الملف
- `NumeroDec` — رقم القرار
- `DateDec` — تاريخ القرار (dd/mm/yyyy)
- `ChambreIds` — معرف الغرفة (1-7)
- `DecisionPriseParId` — قرار صادر بغرفتين أو جميع الغرف (2=غرفتين، 3=مجموع الغرف)
- `Sujet` — الموضوع أو النص القانوني (3 أحرف على الأقل، 50 كحد أقصى)

### نتائج اختبارات البحث
| المصطلح | الغرفة | عدد النتائج |
|---------|--------|------------|
| طلاق | مدنية | 642 |
| سرقة | جنائية | 751 |
| عقار | عقارية | 634 |
| شيك | تجارية | 100 |
| تعويض | مدنية | 4,731 |

### استرجاع نص القرار
- كل قرار له `encryptedId` (مشفر)
- استرجاع النص: `GET /Decisions/GetArret?encryptedId=ENCODED_ID`
- يُرجع ملف PDF مباشرة

### مزايا للوكلاء الذكيين
- ✅ HTML نصي بسيط (استهلاك رموز منخفض)
- ✅ نتائج بحث بـ POST form (لا حاجة لـ JavaScript)

- ✅ نسخة فرنسية متاحة: `/fr/Decisions/RechercheDecisionsFr`

### مسارات مفيدة
- `/ar` — الصفحة الرئيسية (عربي)
- `/Decisions/RechercheDecisions` — صفحة البحث (عربي)

## 🔧 نموذج كود للبحث في juriscassation.cspj.ma

```python
import requests
from urllib.parse import quote

class JurisCassationSearch:
    BASE_URL = "https://juriscassation.cspj.ma"
    
    CHAMBERS = {
        1: "الغرفة المدنية",
        2: "غرفة الأحوال الشخصية والميراث", 
        3: "الغرفة التجارية",
        4: "الغرفة الإدارية",
        5: "الغرفة الاجتماعية",
        6: "الغرفة الجنائية",
        7: "الغرفة العقارية"
    }
    
    def __init__(self):
        self.session = requests.Session()
        self.session.headers.update({
            "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64)"
        })
        self.token = self._get_csrf_token()
    
    def _get_csrf_token(self):
        r = self.session.get(f"{self.BASE_URL}/Decisions/RechercheDecisions")
        import re
        m = re.search(r'value="(CfDJ[^"]*)"', r.text)
        return m.group(1) if m else None
    
    def search(self, subject, chamber_id=None, decision_number=None, 
               file_number=None, date=None, decision_by=None):
        data = {
            "NumeroDos": file_number or "",
            "NumeroDec": decision_number or "",
            "DateDec": date or "",
            "ChambreIds": str(chamber_id) if chamber_id else "",
            "DecisionPriseParId": str(decision_by) if decision_by else "",
            "Sujet": subject,
            "__RequestVerificationToken": self.token
        }
        r = self.session.post(
            f"{self.BASE_URL}/Decisions/RechercheDecisionsRes",
            data=data
        )
        return r.text  # HTML with results
    
    def get_decision(self, encrypted_id):
        r = self.session.get(
            f"{self.BASE_URL}/Decisions/GetArret",
            params={"encryptedId": encrypted_id}
        )
        return r.content  # PDF bytes
```

## 📌 ملاحظات مهمة

1. **CSRF Token**: juriscassation.cspj.ma يتطلب token يتجدد دورياً — يجب تحديثه قبل كل بحث
2. **الترميز**: استخدم UTF-8 encoding للبحث بالعربية (`urllib.parse.quote`)
3. **الحد الأدنى**: البحث بالموضوع يتطلب 3 أحرف على الأقل و50 كحد أقصى
4. **اللغات**: جميع المواقع تدعم العربية، بعضها يدعم الفرنسية والإنجليزية
5. **الاستهلاك**: للحصول على أقل استهلاك للرموز، استخرج فقط عناوين القرارات وأرقامها من HTML، ثم حمّل PDF فقط عند الحاجة
